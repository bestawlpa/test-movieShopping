# Netmovie (React)

โปรเจคนี้เป็นแอปพลิเคชันชอปปิ้งภาพยนตร์ที่พัฒนาด้วย React ผู้ใช้สามารถเพิ่มภาพยนตร์ที่ต้องการลงในตะกร้า และสามารถค้นหารายการหนังที่ต้องการได้

## Tech Stack

- React.js
- CSS (Tailwind)
- Context
- LocalStorage

## install dependencies

- npm install

## รันโปรเจค

- npm run dev
